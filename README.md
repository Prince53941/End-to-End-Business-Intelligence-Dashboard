# 📊 Retail Business Analytics Dashboard (Power BI)

## 🔍 Overview
This project showcases an interactive Power BI dashboard designed to analyze retail business performance. It provides insights into sales, product performance, and customer behavior to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall business performance  
- Identify top-performing products and customers  
- Track monthly sales trends  
- Evaluate profit and profit margins  
- Understand regional and segment-wise performance  

---

## 📌 Key Features
- 📈 Sales Performance Analysis (Total Sales, Monthly Trends)  
- 📦 Product Performance Insights (Top/Least Selling Products, Profitability)  
- 👥 Customer Analysis (Top Customers, Revenue per Customer)  
- 🌍 Regional Analysis (Sales distribution by region)  
- 🎯 Dynamic Filters (Category, Region, Month)  
- 📊 Interactive and visually rich dashboard  

---

## 🛠️ Tools & Technologies
- Power BI  
- Excel / CSV  
- DAX (Data Analysis Expressions)  

---

## 📷 Dashboard Pages
- Business Performance Overview  
- Product Performance Dashboard  
- Customer Sales & Profit Analysis  

---

## 📊 Key Insights
- Furniture category contributes the highest sales  
- High-profit products significantly impact revenue  
- Consumer segment leads in overall sales  
- Regional differences highlight growth opportunities  

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open it in Power BI Desktop  
3. Use filters and visuals to explore insights
